Términos de js

  1. función: Encierra un bloque de código en el cual se realiza una tarea específica, la cual se puede invocar desde otra parte del programa.
  2. let: Se utiliza para reservar una palabra o declarar una variable que puede cambiar.
  3. const: Se utiliza para declarar una variable que no puede cambiar despues de ser creada.

Términos en html

  1. table:Se utiliza para crear una tabla en html.
  2. caption: Para añadir un subtitulo a una tabla.
  3. tr: Define una fila de celdas en una tabla.
